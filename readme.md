# Tutorial for FASTapi

## Install

```sh
pip install 'fastapi[all]'
```

## Clone module

```sh
git clone --recursive https://github.com/maple24/fastapi.git/
```

For already cloned repos, use:

```sh
cd CRQM
git submodule update --init --recursive
```

## Quick Start

```sh
uvicorn main:app --reload
```
